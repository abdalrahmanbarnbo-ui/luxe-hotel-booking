# Luxe Hotel | Premium Booking System 🏨✨

A fully functional, responsive, and modern Hotel Booking System designed to provide an exceptional user experience for browsing and reserving luxury accommodations.

## 🌟 Key Features

* **Modern User Interface:** Built with a focus on premium aesthetics and smooth animations.
* **Fully Responsive:** Flawless experience across desktop, tablet, and mobile devices.
* **Smooth Navigation:** Interconnected sections with smooth scrolling for a seamless Single Page Application (SPA) feel.
* **Dynamic Routing:** Integrated routing for detailed room views, amenities, and contact information.
* **SEO Optimized:** Structured perfectly for search engine visibility.

## 🛠️ Tech Stack

* **Frontend:** React 19, TypeScript
* **Styling:** Tailwind CSS
* **Build Tool & Dev Server:** Vite
* **Icons:** Lucide React
* **Routing:** React Router DOM
